# PastPaperTracker

Simple browser-based tracker for IAL past paper practice.

## Features
- 4 subject options (Mathematics, Physics, Chemistry, Biology)
- Goal-based tracker generation by number of years to solve
- 3 paper series per year: January, May/June, October/November
- Per-paper progress states: Not Done, In Progress, Done, Done + Reviewed
- Live status summary counts

## Run locally
Open `index.html` in a browser.
